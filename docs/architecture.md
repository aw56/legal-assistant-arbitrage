# Архитектура Legal Assistant API

## Общая концепция
- Цифровой юрист по арбитражным делам
- Модульность, автономность, масштабируемость

## Модули
1. Нормативная база
2. Актуализация
3. Клиентский запрос
4. Правовой анализ
5. Коллизии
6. Юридическое заключение
7. Прогноз успеха
8. Субъективные предпосылки
9. Документы
10. Автономность
11. Калькулятор
12. Маркетинг

## Взаимосвязь модулей
- Запрос → Анализ → Заключение → Документ → Прогноз
